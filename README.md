# Personal CV Website
https://imdhani795.github.io/CV_BimaRahmadhani/

A personal CV website built entirely using HTML to showcase my professional profile, technical skills, certifications, educational background, and project experiences. This website was designed as a digital portfolio to present my background in Informatics, highlight my abilities in web development, and demonstrate my interest in technology and continuous learning.

The website contains several sections including personal information, professional summary, work experiences, certifications, technical skills, and project portfolios. It is structured with a clean and responsive layout to provide a simple and professional user experience.

Through this project, I applied fundamental web development concepts using pure HTML while organizing content in a structured and accessible way. The website also reflects my attention to detail, problem-solving abilities, and commitment to creating professional digital solutions.

## Features
- Personal profile and introduction
- Professional summary
- Work and internship experiences
- Technical skills and certifications
- Project portfolio section
- Responsive and clean layout
- Simple and user-friendly interface

## Technologies Used
- HTML5

## Purpose
This project was created to:
- Build a professional online CV and portfolio
- Showcase technical and professional competencies
- Demonstrate basic frontend development skills
- Support career opportunities in technology and information systems
